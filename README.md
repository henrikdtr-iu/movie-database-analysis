# Movie Database Analysis (Python)

This project demonstrates the implementation and analysis of a simple movie database using Python and SQLAlchemy.

## Contents
- Python script for database creation and analysis
- Sample datasets (CSV)
- Reproducible setup

## Requirements

- Python 3.x
- pandas
- SQLAlchemy

## How to run
1. Install dependencies:
   pip install pandas sqlalchemy

2. Run script:
   python streaming_database.py

The database will be created automatically.
